API คือ ข้อมูลที่ผู้ใช้บริการ (เรา) ไปขออนุญาตมาใช้ (การดึง API) มาจากผู้ให้บริการ โดย API จะเป็น raw ข้อมูล โดยเราต้องนำมาจัดการต่อตามการใช้งานของเราเอง โดย API เปรียบเสมือนช่องทางที่ผู้ใช้บริการ สามารถขอข้อมูลบางอย่างจากผู้ให้บริการได้ โดยบางผู้ให้บริการ อาจจำกัดการนำ API ไปใช้งาน
